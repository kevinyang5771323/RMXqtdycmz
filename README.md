# 项目简介

## 核心功能
- 员工管理
- 餐饮菜品管理
- 菜品订单管理
- 门店原材料管理
- 原料进货管理
- 原料出库管理
- 员工考勤管理
- 员工工资管理
- 个人中心
- 餐饮菜品查看
- 员工考勤查看
- 员工工资查看
- 门店原材料查看

## 适用场景
餐饮企业财务管理

## 技术特点
- 架构：B/S、MVC、Web框架
- 开发环境：IDEA、JDK1.8、Mysql、Maven、Navcat
- 技术栈：Java、SpringBoot、Vue
